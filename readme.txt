Homework 4: ReadMe


Commands		Task
thread			runs normal thread.c without any locks
thread_spin		runs thread_spin.c with the implementation of spinlock for 2 threads
thread_m		runs thread_m.c with the implementation of mutex lock for 2 threads
thread_cv		runs thread_cv.c with the implementation of condition variables for 10 threads
thread_ptv		runs thread_ptv.c with the implementation of per thread variables for 10 threads
thread_cm		runs thread_cm.c with the implementation of cute macros for 10 threads
thread_mem		runs thread_mem.c with the fix of sbrk and malloc for 10 threads
